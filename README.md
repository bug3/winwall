# winwall

winwall is a tool to get wallpaper on Windows 10/11 lock screen.

## Requirements

- Windows 10 / 11
- [Go](https://go.dev/)

## Build

Run this command:

```batch
go build -o winwall.exe
```
or run **build.bat** file.

## Usage
If build is successful run **winwall.exe** file.

Then the wallpapers will be on the desktop.
